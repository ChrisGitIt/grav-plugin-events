\EventsEvents

Events Plugin Events Class

The Events Class instantiates an instance of all Events available in Grav and provides filters for filtering out uneeded events, setting offsets, limits and etc. This creates a listing of events applies repeat rules, for example MTWRFSU, frequency rules, for example daily, weekly, monthly, or yearly, creates event instances with just date information tied to a 6 digit token, and then it filters down and adds an events that need added to Grav Pages.

all()

all() : object

Get an instance of events

This is the bulkhead of this class. It creates an events listing using various rules and params to add events to the page list while also adding all events, including repeating, to an events array with a searchable token. The events array contains the appropriate date information for use in templates
